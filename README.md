# flask-vuejs-startup-kit
Python Flask &amp; Vuejs SPA(Single Page App) project template

## Features
- Backend
    - Python3.x
    - Flask
    - Flask JWT-extended
- Frontend
    - node (v12.13.1)
    - npm (v6.12.1)
    - TypeScript(v3.5.3)
    - SCSS(v4.12.0)
    - Vuejs (cli v4.0.5)
    - Buefy
    - Bulma

## Development
### Start up local server
### Frontend
``` sh
cd /path/to/flask-vuejs-startup-kit/frontend
npm run serve
```

### Backend
``` sh
cd /path/to/flask-vuejs-startup-kit
python3 -B dev.py
```
